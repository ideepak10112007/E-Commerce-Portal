This project is a single-page E-Commerce demo named LuciDeep, built using only HTML, CSS, and vanilla JavaScript (all in one file). 
It showcases a modern storefront with category-based product browsing (tech, groceries, appliances, fashion), real-time product images from the web, and fully client-side functionality using `localStorage`â€”so it runs without any backend.

Key features

* Sticky navigation: Home, Products, Orders, Wishlist, Reviews, Customer Care, Reporting, About, Contact
* Form validation for Registration/Login and Contact/Support forms (email, phone, password rules)
* Cart & Wishlist with quantity controls and order placement (demo flow)
* Orders list with status and timestamps; **CSV export** for reporting (users, orders, revenue)
* Reviews system with ratings and comments
* Customer Care ticket form with basic SLA info
* Responsive UI and accessible controls; images open in new tabs; optional outbound product links.
